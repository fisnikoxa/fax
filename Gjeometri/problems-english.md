## 1. Draw the three projections of points E and F. The distances of point E should be equal from the projection planes:
   - a) Point E should be in the first octant and point F symmetric to point E with respect to plane π₃
   - b) Point E should be in the fourth octant and F symmetric to point E with respect to plane π₂

**Solution:**
- a) If E has equal distances from all three projection planes: E(3, 3, 3). Symmetry with respect to π₃: F(-3, 3, 3)
  - Projections: E'(3,3), E''(3,3), E'''(3,3); F'(-3,3), F''(-3,3), F'''(3,3)
- b) In octant IV with equal distances: E(3, -3, 3). Symmetry with respect to π₂: F(3, 3, 3)
  - Projections: E'(3,-3), E''(3,3), E'''(-3,3); F'(3,3), F''(3,3), F'''(3,3)

## 2. Draw the projections and determine in which octant each of the points A(3, 2, 4), B(2, -3, 2), C(-2, 5, 4), D(-4, -2, 5), E(-4, -2, -3) is located!

**Solution:**
- A(3, 2, 4): (+, +, +) → Octant I
- B(2, -3, 2): (+, -, +) → Octant IV
- C(-2, 5, 4): (-, +, +) → Octant II
- D(-4, -2, 5): (-, -, +) → Octant III
- E(-4, -2, -3): (-, -, -) → Octant VII

## 3. Given point A(A", A'''). Determine the first projection A' of point A, when:
   - a) When A is located in octant I,
   - b) A is located in octant II!

**Solution:**
If given A''(x, z) and A'''(y, z), then:
- a) In octant I (x > 0, y > 0, z > 0): A' = (x, y) where x > 0 and y > 0
- b) In octant II (x < 0, y > 0, z > 0): A' = (x, y) where x < 0 and y > 0
Example: If A''(3, 4) and A'''(2, 4), then:
- a) A in octant I: A(3, 2, 4) → A'(3, 2) in quadrant I (bottom right)
- b) A in octant II: A(-3, 2, 4) → A'(-3, 2) in quadrant IV (bottom left)

## 4. Draw the three projections of segment AB and find its true length when given:
   - a) A(1, 3, 2), B(5, 1, 1),
   - b) A(-1, 2, 3), B(2, 2, 2),
   - c) A(2, -3, -4), B(2, -1, 2)!

**Solution:**
Formula: |AB| = √[(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²]
- a) A(1, 3, 2), B(5, 1, 1): Δx=4, Δy=-2, Δz=-1 → |AB| = √21 ≈ 4.58 units
- b) A(-1, 2, 3), B(2, 2, 2): Δx=3, Δy=0, Δz=-1 → |AB| = √10 ≈ 3.16 units (parallel to π₂)
- c) A(2, -3, -4), B(2, -1, 2): Δx=0, Δy=2, Δz=6 → |AB| = √40 ≈ 6.32 units (parallel to π₁)

- A₃ (π₃, x=0): t=1.5 → A₃(0, 4.5, 2)
Angles of inclination (|d| = 3):
- α₁ ≈ 41.8° (with π₁)
- α₂ ≈ 19.5° (with π₂)
- α₃ ≈ 41.8° (with π₃)

## 9. Through point P(3, 4, 2) construct the line that will intersect line a=EF[E(-2, 3, 2), F(2, 1, 6)] at the point located in plane π₁.

## 10. Through point P(3, 4, 2) construct line b parallel to plane π₂ that intersects line a=LK[L(-2, 3, 2), K(2, 1, 6)]!

**Solution:**

- Vector: d = (-7, 0, -2). Line b: (3,4,2) + t(-7,0,-2)

## 11. Through point P(0, 2, 4) construct the projections of line b which intersects line a=MN[M(3, -2, -3), N(3, 1, 5)] and is parallel to plane π₁!

**Solution:**
Parallel to π₁ means z constant. Line b: (0,2,4) + t(a,b,0)
- Line a: (3,-2,-3) + s(0,3,8). Note: x=3 constant (parallel to π₃)
- For z=4: -3+8s=4 → s=7/8 → Q(3, 0.625, 4)
- Line b passes through P(0, 2, 4) and Q(3, 0.625, 4)
- Vector: d = (3, -1.375, 0). Line b: (0,2,4) + t(3,-1.375,0)
- P(0,2,4) is B₃. No B₁ (parallel to π₁)
